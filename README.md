# AI Репетитор

Сайт для учнів 3 класу. AI дає підказки, не каже відповідь. Перевіряє чи правильно порахував.

## Структура папки

```
ai-tutor-site/
├── index.html          # Головна сторінка
├── style.css           # Стилі
├── app.js              # Логіка чату
├── package.json        # Залежності для Vercel
├── .gitignore
├── api/
│   └── chat.js         # Backend (OpenAI)
├── README.md
└── VERCEL-ІНСТРУКЦІЯ.md   # Покрокова інструкція
```

## Швидкий старт

**Дивись файл `VERCEL-ІНСТРУКЦІЯ.md`** — там покроково все описано.

Коротко:
1. Отримай ключ на platform.openai.com
2. Залей цю папку на GitHub
3. Підключи до Vercel, додай змінну OPENAI_API_KEY
4. Deploy
