# Покрокова інструкція: Як задеплоїти сайт на Vercel

## Крок 1. Отримай OpenAI API ключ

1. Зайди на **https://platform.openai.com**
2. Зареєструйся або увійди
3. Натисни на свій профіль (у правому верхньому куті) → **View API keys**
4. Натисни **Create new secret key**
5. Скопіюй ключ (він починається з `sk-`). **Збережи його** — більше він не покажеться!

---

## Крок 2. Залей проєкт на GitHub

### Якщо ще не маєш Git:
- Завантаж Git: https://git-scm.com/download/win
- Встанови й перезавантаж комп'ютер

### Створення репозиторію:

1. Зайди на **https://github.com**
2. Увійди в акаунт (або створи новий)
3. Натисни **+** (у правому верхньому куті) → **New repository**
4. **Repository name:** `ai-tutor` (або будь-яка назва)
5. Обери **Public**
6. **НЕ** став галочку "Add a README file"
7. Натисни **Create repository**

### Завантаження файлів:

**Варіант А — через веб-інтерфейс GitHub:**
1. На сторінці нового репо натисни **uploading an existing file**
2. Перетягни в браузер **всі файли** з папки `ai-tutor-site`:
   - index.html
   - style.css
   - app.js
   - package.json
   - .gitignore
   - папку **api** (з файлом chat.js всередині)
3. Внизу сторінки напиши: `Перший коміт`
4. Натисни **Commit changes**

**Варіант Б — через термінал:**
```
cd C:\Users\soeas\ai-tutor-site
git init
git add .
git commit -m "AI Репетитор"
git branch -M main
git remote add origin https://github.com/ТВІЙ_ЛОГІН/ai-tutor.git
git push -u origin main
```
(заміни `ТВІЙ_ЛОГІН` на свій логін GitHub)

## Крок 3. Підключи Vercel і задеплой

### 3.1 Реєстрація на Vercel
1. Зайди на **https://vercel.com**
2. Натисни **Sign Up**
3. Обери **Continue with GitHub** — увійдеш через GitHub
4. Дозволь Vercel доступ до твоїх репозиторіїв (натисни **Authorize**)

### 3.2 Імпорт проєкту
1. На головній Vercel натисни **Add New...** → **Project**
2. Знайди свій репозиторій **ai-tutor** (або як ти його назвав)
3. Натисни **Import** біля нього

### 3.3 Налаштування — дуже важливо!
Перед деплоєм:
1. **Framework Preset** — залиш **Other** (або Vite, не важливо)
2. Прокрути вниз до **Environment Variables**
3. У полі **Key** введи: `OPENAI_API_KEY`
4. У полі **Value** встав свій ключ з OpenAI (той що `sk-...`)
5. Переконайся, що обрано **Production**, **Preview**, **Development** (можна всі три)
6. Натисни **Add**

### 3.4 Деплой
1. Натисни синю кнопку **Deploy**
2. Почекай 1–2 хвилини (буде показувати прогрес)
3. Коли з'явиться **Congratulations!** — готово!

### 3.5 Отримання посилання
- Натисни **Visit** — відкриється твій сайт
- Або скопіюй посилання типу: `https://ai-tutor-xxxx.vercel.app`

## Крок 4. Готово!

Тепер сайт працює онлайн. Можеш:
- Відкрити його на телефоні
- Додати в закладки
- Зробити додаток через конструктор (AppsGeyser тощо) — вказати це посилання

---

## Якщо щось не працює

**Помилка "Додай OPENAI_API_KEY":**
- Зайди на vercel.com → свій проєкт → **Settings** → **Environment Variables**
- Перевір, що змінна `OPENAI_API_KEY` є і містить правильний ключ
- Зроби **Redeploy**: вкладка **Deployments** → три крапки біля останнього деплою → **Redeploy**

**Сайт не відкривається:**
- Перевір посилання (має бути https://)
- Спробуй інший браузер
